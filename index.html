Ugliest Website UI: Home - Yale School of Art

What I would do to improve it:

  • Simplify the layout by having a consistent color scheme and reduce the colors used. Aligning the text and images using grid would also simplify the visual clutter.
  • Improve user experience by adding a navigation bar at the top.
  • Change the website to only use 2 fonts, one for the headers and one for the content texts. Also remove the text element background colors and adjust the color to be contrasting with the background to make it readable.
  • Separate the different sections within the page with whitespace.
  • Reduce the main page information to only the necessary ones. Add more pages where more informations about the subtopics will be put on.
  • Replace the background image with something relevant to the school or just make it a static white color to make it look clean.
